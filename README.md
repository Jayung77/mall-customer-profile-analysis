# 🛒 쇼핑몰 회원 프로파일 및 구매 전환 보틀넥 분석 (SQL 포트폴리오)

> **Kaggle 고객 데이터**를 MySQL을 통해 구조화하고, 가입 추이 및 데모그래픽 정밀 분석을 거쳐 이탈 방지 및 타깃 프로모션 전략을 수립한 프로젝트입니다.

## 🛠️ 사용 기술 스택
* **Data Processing & Analysis:** SQL / MySQL (MySQL Workbench)
* **Visualization:** Excel (엑셀 시각화)
---

## 📌 1. 배경 및 목표 (Overview)
* **분석 목적:** 쇼핑몰의 실제 회원 행동 데이터를 분석하여 장기 미접속자 리마케팅 설계 기반을 마련합니다.
* **핵심 과제:** 연령대별/세그먼트별 상이하게 작용하는 가입 및 구매 요인을 탐색하여 맞춤화된 타깃 프로모션 전략을 제안합니다.
* **분석 프로세스:** 가입년월별 회원수 추이 파악 ➔ 성별/연령 분포 분석 ➔ 구매 전환율 검증 (보틀넥 영역 식별)

---

## 🗄️ 2. 데이터 마트 구축 (Data Processing)
분석의 효율성을 위해 `CUSTOMER` 정보에 `SALES` 테이블을 `LEFT JOIN`하여 단일 통합 데이터 마트(`CUSTOMER_PROFILE`)를 설계했습니다.
* **변수 가공:** 가입년월(%Y-%m) 가공, 기준년도(2021년) 기반 나이 연산 및 연령대 조건 분류 (10대 이하 ~ 50대 이상)
* **구매 여부 정의:** 판매 데이터가 존재하면 '구매', 부재하면 '미구매'로 명시

---

## 🔍 3. 핵심 분석 결과 및 비즈니스 인사이트

### 📈 시계열 분석: 가입년월별 회원수 추이
* **성장 이상 현상 발견:** 2019년 4월(2명) 대비 **2019년 10월(312명)에 가입자가 전례 없이 폭증하는 Peak 구간**을 확인했습니다.
* **전략 제안:** 단기간에 트래픽이 대폭 급증한 시점의 내부 마케팅 프로모션 및 외부 검색 유입 채널을 역추적하여, 성공 전략을 타 분기 캠페인에 템플릿화할 것을 제안합니다.

### 📊 데모그래픽 분석: 성별 및 연령대별 분포
* **압도적인 특정 세그먼트 편중:** 여성 평균 나이 35.1세, 남성 평균 나이 36.9세로 균등하나, **30대 여성 회원수는 총 756명으로 남성 대비 약 2.5배 이상의 초격차 현상**을 보입니다.
* **전략 제안:** 서비스의 주력 엔진인 30대 여성층을 위해 메인 배너 및 추천 상품 섹션을 해당 세그먼트의 최선호 카테고리 위주로 고도화할 필요가 있습니다.

### 🚨 핵심 발견: 구매 전환의 보틀넥(Bottleneck) 식별
* **가설 검증 결과:** 가장 볼륨이 큰 **30대 여성층(756명) 중 실제 구매자는 단 100명에 그쳤으며, 나머지 656명은 미구매 이탈 상태**인 심각한 보틀넥을 식별했습니다.
* **원인 분석:** 가입 유입 마케팅은 성공적이었으나, 유입 이후 '장바구니' 혹은 '상세 탐색' 단계에서 가격 장벽이나 매력적인 상품 라인업 부재로 인해 중도 이탈했을 가능성이 매우 높습니다.

## 🎯 4. 실행 전략 제안 (Strategic Plan)

1. **웰컴 혜택 설계 (이탈 최소화)**
   * 가입 후 구매 전환이 안 된 30대 여성 이탈 회원을 타깃으로 '첫 구매 유도 웰컴 쿠폰북' 및 기간 한정 타임특가 마케팅 자동화 파이프라인 구축을 제안합니다.
2. **맞춤 기획전 편성 (고객 활성화)**
   * 30대 여성이 가입 직후 장바구니에 가장 빈번하게 담는 인기 상위 카테고리 데이터를 모니터링하여 메인 화면 전면부에 맞춤형 추천 기획전을 고정 배치합니다.
3. **마케팅 캠페인 역추적**
   * 최고점을 기록했던 2019년 10월의 프로모션 유입 경로를 추적 분석하여 양질의 트래픽을 지속 수급할 수 있도록 매체 구조를 최적화합니다.
