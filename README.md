# mall-customer-profile-analysis
쇼핑몰 회원 프로파일 분석 및 구매 전환율 개선을 위한 SQL 데이터 분석

# 쇼핑몰 회원 데이터 분석 프로젝트

## 1. 프로젝트 목적
쇼핑몰의 회원 데이터를 분석하여 가입 추이와 연령·성별 분포를 파악하고, 실제 구매 여부를 분석하여 마케팅 전략을 제안합니다.

## 2. 사용 기술
* 데이터 추출 및 가공: MySQL, SQL Workbench
* 데이터 시각화: Excel (엑셀)

## 3. 핵심 분석 내용
* 가입년월별 회원수 추이 분석 (가입자 급증 원인 파악)
* 성별 및 연령대별 회원수 분포 추출
* 성별/연령대별 실제 구매 여부 비율 분석 (구매 전환율 진단)

## 4. 분석 결과 및 전략 제안
* **현황:** 30대 여성 회원수가 가장 많지만, 실제로 구매까지 이어진 비율은 매우 낮습니다. (유입은 성공했으나 이탈이 심한 상태)
* **전략:** 30대 여성 고객의 이탈을 막기 위해 '첫 구매 웰컴 쿠폰'을 지급하거나, 해당 연령대 맞춤형 기획전을 메인 화면에 노출하는 전략이 필요합니다.
